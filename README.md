# RemarkableBecause
Our website for #IAmRemarkable

## About the Project
The site is inspired by Google's #IAmRemarkable initiative which aims employers to do self-promotion. 

## Tools 🛠️
Google App Engine, AJAX + Datastore, Healthruwords API, HTML/CSS, and Python
